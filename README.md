# Ercat
<hr>
<h5>>> Ercat is a simple HTTP server made with C.</h5>
<h2>Creator: ProX</h2>
<hr>
<h3>Installation:</h3>
| git clone https://github.com/Pr00x/ercat<br>
| sudo make install
<hr>
<h3>Usage:</h3><br>
| ercat -p port<br><br>
For Example: sudo ercat -p 80<br>
